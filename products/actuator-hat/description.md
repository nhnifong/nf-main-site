## Damiao actuator connectivity

This raspberry pi zero hat provides an interface to the relatively uncommon XT60(2+2) connector used by Damiao actuators.
Combine this with a raspberry pi zero 2w and a suitable 24v supply and you create a variety of two actuator robot systems.

## Connectivity and Specs

- **Input Power:** 24V 5.5x2.1 barrel jack power input
- **Raspberry Pi Interface:** Raspberry Pi Zero hat format. Provides stable 5v to the pi.
- **CAN bus:** A CAN bus interface is provided between the raspberry pi and the motors
- **Power take off:** Extra sockets are provided for taking off 24v or 5v for other purposes.
